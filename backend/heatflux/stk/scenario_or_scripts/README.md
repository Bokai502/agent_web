# STK Execution Note

STK Desktop/Engine and a valid license were not detected in the current Linux
environment, so the STK scenario could not be executed here.

Recommended STK setup for a licensed Windows or STK Engine environment:

1. Create one scenario per analysis date: March 21, June 21, September 21, and
   December 21.
2. Create CATCH02 as a 600 km circular Sun-synchronous satellite.
3. Use LTAN 06:00 for dawn-dusk and LTAN 12:00 for noon cases.
4. Export satellite ECI position/velocity, Sun vector, eclipse state, and
   attitude or body-frame face normal vectors at 30 s cadence for at least one
   orbit, preferably 24 h.
5. Feed the exported CSV into the same heat-flux postprocessor used for the
   Python/Orekit baseline.

Current reproduction outputs were generated by the open Python baseline.
