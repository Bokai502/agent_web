# 分类方法论

## 步骤1: 关键词识别与初步定位
分析元器件名称中的关键词、缩写、技术特征，确定其所属的大类和功能类别。

**关键词映射规则：**

### 集成电路类关键词
- **数字逻辑**：TTL、ECL、CMOS、74系列、CD4000系列 → 数字集成电路
- **模拟功能**：运放、LM、OP、比较器、调整器、稳压、PWM、开关 → 模拟集成电路
- **处理器**：CPU、MCU、DSP、单片机、STM32、ARM、AVR → 微处理器
- **存储**：RAM、ROM、Flash、EEPROM、SRAM、DRAM → 存储器
- **可编程**：FPGA、CPLD、GAL、PAL → 可编程门阵列
- **转换器**：ADC、DAC、A/D、D/A → AD/DA转换器
- **接口**：RS232、RS485、CAN、SPI、I2C、UART → 外围接口电路
- **电平转换**：电平转换、Level Shifter → 电平转换器
- **霍尔**：霍尔、Hall → 霍尔集成电路
- **微波/射频IC**：微波集成、MMIC、射频IC → 微波、超高频集成电路

### 半导体分立器件关键词
- **二极管通用**：二极管、Diode、整流、检波
  - 检波：检波二极管
  - 整流/硅整流：硅整流二极管
  - 开关：开关二极管
  - 稳压/齐纳/Zener：电压基准二极管
  - 肖特基/Schottky：肖特基二极管
  - TVS/瞬态抑制：瞬态电压抑制二极管
  - 变容/Varactor：变容二极管
  - 混频：检波、混频二极管
  
- **晶体管通用**：三极管、晶体管、BJT、Transistor
  - 高频/RF：中小功率高频晶体管
  - 开关/Switch：中小功率开关晶体管或大功率开关晶体管（根据功率）
  - 低频/音频：大功率低频晶体管
  - 达林顿/Darlington：达林顿晶体管
  - 微波：微波晶体管
  
- **场效应管**：
  - MOSFET、MOS管、N沟道、P沟道 → MOS型场效应晶体管
  - JFET、结型场效应 → 结型场效应晶体管

### 频率控制类关键词
- **晶体**：晶振、晶体、石英、Crystal、XTAL
  - 有源晶振/OSC/振荡器 → 晶体振荡器
  - 无源晶振/谐振器 → 晶体谐振器
- **非晶体振荡器**：陶瓷振荡器、RC振荡器、LC振荡器 → 非晶体振荡器

### 滤波器类关键词

### 电阻器类关键词
- 金属膜电阻 → 金属膜固定电阻器
- 金属氧化膜 → 金属氧化膜固定电阻器
- 线绕电阻 → 线绕固定电阻器

## 步骤2: 层级匹配与精确定位
根据步骤1识别的大类，在对应的层级体系中进一步细化匹配：

1. **确定第一级别**：判断属于哪个大类（集成电路、半导体分立器件、谐振器等）
2. **确定第二级别**：在大类下确定具体子类（如集成电路下的单片/混合，二极管下的普通/微波）
3. **确定第三级别**：进一步细化功能类别（如单片集成电路下的数字/模拟/存储等）
4. **确定最终分类**：匹配到83个标准名称中的一个

## 步骤3: 特殊情况处理

### 模糊或通用名称处理
- 如果名称过于模糊，选择"其他"或"其他"类别：
  - 模拟IC → 其他模拟集成电路
  - 滤波器（未指明类型）→ 其他滤波器
  - 连接器（未指明类型）→ 其他电连接器
  - 微波器件（未指明类型）→ 其他微波器件
  - 微波二极管（未指明类型）→ 其他微波二极管

### 功能重叠处理
- 在上述83个标准分类名称中查找**最精确匹配**的分类
- 优先选择功能最具体、描述最准确的分类
- 注意区分相似分类（如"晶体谐振器"vs"晶体振荡器"）

## 步骤3: 严格验证输出
- **必须验证**：输出的分类名称是否在83个标准名称中**完全一致**（包括标点符号、括号、顿号等）
- **绝对禁止**输出列表之外的任何分类名称
- **绝对禁止**修改、简化或创造新的分类名称

# 重要约束
1. **严格匹配**：输出的分类名称必须与83个标准名称中的某一项**完全一致**，包括所有汉字、标点符号、括号和顿号
2. **一对一映射**：每个元器件名称只能对应一个分类名称
3. **最佳匹配原则**：
   - 优先匹配功能最具体的分类
   - 如果有多个可能的分类，选择技术特征最接近的
   - 当无法确定时，选择更通用的分类（如"其他模拟集成电路"、"其他滤波器"、"其他微波器件"、"其他电连接器"等）
4. **完整输出**：必须为输入的每个元器件名称都提供对应的分类
5. **禁止猜测**：如果元器件名称非常模糊且无法合理推断，选择最接近的"其他"类别
